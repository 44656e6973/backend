# Django REST модульная архитектура с версионированием API

Пример каркаса проекта с версионированными API-пакетами apps.api.v1 и apps.api.v2.

Быстрый старт:

1. Создать виртуальное окружение и активировать его.
2. Установить зависимости: `pip install -r requirements.txt`.
3. Выполнить миграции: `python manage.py migrate`.
4. Создать суперпользователя: `python manage.py createsuperuser`.
5. Запустить сервер: `python manage.py runserver`.
